# 学生成绩管理系统

中学教务成绩管理系统，支持多学科成绩录入、等级评定、多线程并发安全、成绩持久化、批量统计与偏科分析。

---

## 目录结构

```
exam_system/
├── main.py              # 程序入口，整合所有模块
├── grade_utils.py       # 通用工具函数模块
├── exam_records.txt     # 成绩记录文件（运行时自动生成）
├── README.md            # 项目说明文档
└── subjects/            # 学科包
    ├── __init__.py      # 包标识 + 统一导入
    ├── base_exam.py     # 考试抽象基类
    ├── chinese.py       # 语文学科子类
    ├── math.py          # 数学学科子类
    └── english.py       # 英语学科子类
```

---

## 快速开始

```bash
cd exam_system
python main.py
```

---

## 功能概览

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 学科信息管理 | 每门学科有名称、满分值、学生答卷 |
| F2 | 成绩录入 | 录入分数后自动验证合法性，超出满分拒绝录入 |
| F3 | 等级评定 | 不同学科有各自的等级划分规则 |
| F4 | 成绩持久化 | 每次录入记录写入 `exam_records.txt` |
| F5 | 多线程安全录入 | 多名老师并发录入时数据不冲突 |
| F6 | 批量统计 | 传入多份答卷实例，遍历计算加权分（多态） |
| F7 | 优秀筛选 | 列表推导式筛选达到优秀的学生 |
| F8 | 成绩单生成器 | 使用 yield 惰性生成学生成绩单 |

## 各学科等级规则

| 学科 | 类名 | 满分 | 等级规则 |
|------|------|------|----------|
| 语文 | `ChineseExam` | 150分 | ≥135优秀，≥120良好，≥90及格，<90不及格 |
| 数学 | `MathExam` | 150分 | ≥140优秀，≥120良好，≥90及格，<90不及格 |
| 英语 | `EnglishExam` | 100分 | ≥90优秀，≥75良好，≥60及格，<60不及格 |

---

## 额外特性

- **语文**：独有属性 `essay_score`（作文分，满分60）
- **数学**：私有属性 `__bonus_points`（附加分，默认0，上限15），加权分计算包含附加分
- **英语**：私有属性 `__listen_score` / `__reading_score` / `__writing_score`（听力/阅读/写作分项成绩），重写 `print_report_card()` 打印分项成绩单

## 进阶功能

| 挑战 | 函数/方法 | 说明 |
|------|-----------|------|
| 班级排名 | `calculate_class_ranking()` | 按总分降序排列学生 |
| 成绩进步追踪 | `compare_with_previous()` | 对比上次成绩，输出差值、变化百分比、等级变化 |
| 偏科判断 | `check_balance()` / `calculate_standard_deviation()` | 统一折算为百分制后计算标准差，判断是否偏科 |
| 异常处理 | `try...except` 全覆盖 | 所有可能异常的操作均有友好错误提示 |

---

## 设计要点

- **面向对象**：使用 `ABC` 抽象基类 + 继承 + `super().__init__()` + 多态
- **私有属性**：成绩 `__score`、英语分项成绩、数学附加分均使用双下划线私有化，通过 getter/setter 访问
- **线程安全**：全局 `student_records` 字典 + `threading.Lock()` 互斥锁保护并发写入
- **文件操作**：所有文件读写均使用 `with` 上下文管理器，编码统一为 UTF-8
